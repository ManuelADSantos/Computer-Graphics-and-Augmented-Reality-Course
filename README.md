# Computer Graphics and Augmented Reality Course
This repository contains the course materials, assignments, and projects for the Computer Graphics and Augmented Reality course offered at University of Coimbra. 

The course covers fundamental concepts in computer graphics, 3D modeling, rendering techniques, and augmented reality applications.
